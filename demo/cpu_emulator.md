简易的cpu指令模拟，初始输入指令对应的二进制代码，然后开始执行。

界面如：
```
|----------------|             
|                | |---------| 
|  bincode input | |load btn |
|                | |---------|
|----------------|

|----------------| |---------------|
|                | |  cpu state    |
|                | |---------------|
| memory         | 
|                | |---------------|
|                | |  instruction  |
|                | |               |
|----------------| |---------------|

|------------------| |-------| 
|  next_inst       | | reset | 
|------------------| |-------|
```

https://masswerk.at/6502/assembler.html

输入的二进制代码是十六进制数字，忽略空格
